
5. [Drawing a Shape](https://youtu.be/t_ispmWmdjY?t=721)
'print' does not include a new line, 'puts' does include a new line.
```print "hello"
puts "world"
puts "hiya"
```
Output
```Hello Rubyhelloworld
hiya
```
Basic shape,
```puts "    /|"
puts "   / |"
puts "  /  |"
puts " /___|"
```
